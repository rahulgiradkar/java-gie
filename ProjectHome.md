Aplicación web basada en tecnologías java para administrar una empresa informática.
### Tecnologías ###
  1. Jboss 7 AS Web Profile | Apache Tomcat 7
  1. JPA 2.0 (Hibernate 3.6)
  1. Spring Framework 3.0.5
  1. Spring Security 3.1.RC3
  1. JSF 2.0
  1. Primefaces 3.0 (lib componentes JSF)
  1. Apache Tomahawk 2.0.1 (lib componentes JSF)
  1. RBDMS Postgresql 9.1
  1. IDE Eclipse Indigo JEE + JBoss Tools
